# 问题列表接口接入指南

## 📡 接口信息

### 接口地址
```
GET /app/content/question/list
```

### 完整 URL
```
http://123.144.149.59:30560/qa-hero-content/app/content/question/list
```

### 请求方法
```
GET
```

### 认证方式
```
Bearer Token（必须）
```

**请求头**：
```
Authorization: Bearer YOUR_TOKEN
Content-Type: application/json
```

⚠️ **重要**：此接口需要用户认证，必须在请求头中携带有效的 Token。

## 📋 请求参数

### 参数结构

```javascript
{
  pageNum: 1,        // 页码（必填）
  pageSize: 20,      // 每页数量（必填）
  question: {        // 筛选条件（可选）
    userId: 10001,     // 提问者ID
    type: 0,           // 问题类型：0=公开问题，1=悬赏问题，2=定向问题
    categoryId: 5,     // 类别ID
    status: 1,         // 状态：0=草稿，1=已发布，2=已关闭
    title: "如何",     // 标题（模糊搜索）
  }
}
```

### 参数说明

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| pageNum | number | 是 | 页码，从 1 开始 |
| pageSize | number | 是 | 每页数量，建议 10-50 |
| question | object | 否 | 筛选条件对象 |
| question.userId | number | 否 | 提问者ID，筛选指定用户的问题 |
| question.type | number | 否 | 问题类型：0=公开，1=悬赏，2=定向 |
| question.categoryId | number | 否 | 类别ID，筛选指定分类的问题 |
| question.status | number | 否 | 状态：0=草稿，1=已发布，2=已关闭 |
| question.title | string | 否 | 标题关键词，支持模糊搜索 |

/**
 * 获取问题列表
 */
getQuestions: (params) => {
  const { pageNum = 1, pageSize = 20, question = {} } = params;
  
  const requestParams = {
    pageNum,
    pageSize,
    question: {
      ...question,
      params: question.params || {},
    },
  };
  
  return contentApiClient.get(API_ENDPOINTS.QUESTION.LIST, { 
    params: requestParams 
  });
},

/**
 * 获取推荐列表
 */
getRecommendList: (params) => {
  return questionApi.getQuestions({
    pageNum: params.pageNum,
    pageSize: params.pageSize,
    question: {
      status: 1, // 只获取已发布的问题
    },
  });
},

import questionApi from '../services/api/questionApi';

### 步骤 2: 添加数据加载函数

```javascript
/**
 * 加载问题列表
 */
const loadQuestions = async (pageNum = 1, isRefresh = false) => {
  try {
    if (isRefresh) {
      setRefreshing(true);
    } else if (pageNum === 1) {
      setLoading(true);
    } else {
      setLoadingMore(true);
    }
    
    console.log(`🔄 加载问题列表: 页码=${pageNum}, Tab=${activeTab}`);
    
    // 根据当前 Tab 调用不同接口
    let response;
    
    switch (activeTab) {
      case t('home.recommend'):
        response = await questionApi.getRecommendList({ 
          pageNum, 
          pageSize: 20 
        });
        break;
        
      case t('home.hotList'):
        response = await questionApi.getHotList({ 
          pageNum, 
          pageSize: 20 
        });
        break;
        
      case t('home.follow'):
        response = await questionApi.getFollowList({ 
          pageNum, 
          pageSize: 20 
        });
        break;
        
      default:
        // 默认使用推荐列表
        response = await questionApi.getRecommendList({ 
          pageNum, 
          pageSize: 20 
        });
    }
    
    console.log('📥 接口响应:', response);
    
    if (response.code === 200 && response.data) {
      const newData = response.data.rows || [];
      
      console.log(`✅ 获取到 ${newData.length} 条数据`);
      
      // 数据转换：将 API 返回的字段映射到前端使用的字段
      const transformedData = newData.map(item => ({
        id: item.id,
        title: item.title,
        type: item.type === 1 ? 'reward' : item.type === 2 ? 'targeted' : 'free',
        reward: item.bountyAmount ? item.bountyAmount / 100 : 0, // 分转元
        likes: item.likeCount || 0,
        answers: item.answerCount || 0,
        views: item.viewCount || 0,
        location: item.location || '',
        time: formatTimeFromISO(item.createTime),
        // 其他字段可以根据需要添加
        author: '用户' + item.userId, // 临时方案，后续需要获取用户信息
        avatar: `https://api.dicebear.com/7.x/avataaars/svg?seed=user${item.userId}`,
      }));
      
      if (pageNum === 1) {
        setQuestionList(transformedData);
      } else {
        setQuestionList(prev => [...prev, ...transformedData]);
      }
      
      setPage(pageNum);
      setHasMore(newData.length === 20);
    } else {
      console.warn('⚠️ 接口返回异常:', response);
      // 如果是第一页且没有数据，显示空状态
      if (pageNum === 1) {
        setQuestionList([]);
      }
      setHasMore(false);
    }
  } catch (error) {
    console.error('❌ 加载问题列表失败:', error);
    
    // 显示错误提示
    if (error.message) {
      // 可以使用 Toast 或 Alert 显示错误
      console.error('错误信息:', error.message);
    }
    
    // 如果是第一页加载失败，可以显示错误状态
    if (pageNum === 1) {
      setQuestionList([]);
    }
  } finally {
    setLoading(false);
    setRefreshing(false);
    setLoadingMore(false);
  }
};

/**
 * 格式化 ISO 时间为相对时间
 */
const formatTimeFromISO = (isoTime) => {
  const now = new Date();
  const time = new Date(isoTime);
  const diff = now - time;
  
  const minutes = Math.floor(diff / (1000 * 60));
  const hours = Math.floor(diff / (1000 * 60 * 60));
  const days = Math.floor(diff / (1000 * 60 * 60 * 24));
  
  if (days >= 1) {
    return `${days}天前`;
  } else if (hours >= 1) {
    return `${hours}小时前`;
  } else if (minutes >= 1) {
    return `${minutes}分钟前`;
  } else {
    return '刚刚';
  }
};
```

## ⚠️ 注意事项

### 1. 金额单位转换

API 返回的 `bountyAmount` 单位是**分**，需要除以 100 显示为元：

```javascript
reward: item.bountyAmount ? item.bountyAmount / 100 : 0
```

### 2. 时间格式转换

API 返回的是 ISO 8601 格式，需要转换为相对时间：

```javascript
time: formatTimeFromISO(item.createTime)
```

### 3. 用户信息

API 只返回 `userId`，不返回用户名和头像，需要：
- 方案 1: 调用用户信息接口获取
- 方案 2: 后端在问题列表接口中返回用户信息（推荐）

### 4. 问题类型

API 返回的 type 是数字，需要转换：
- 0 → 'free' (公开问题)
- 1 → 'reward' (悬赏问题)
- 2 → 'targeted' (定向问题)

### 5. 空数据处理

当接口返回空数据时，需要显示空状态提示。

### 6. 错误处理

需要处理各种错误情况：
- 网络错误
- 接口错误
- 数据格式错误
