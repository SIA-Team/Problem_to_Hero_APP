## 如何添加新的真实接口

当后端完成新的接口后，在 `src/config/env.js` 中添加配置：

```javascript
const API_MODE_CONFIG = {
  // ... 现有配置 ...
  
  // 新完成的接口
  '/questions': 'real',           // 问题列表
  '/questions/hot': 'real',       // 热门问题
  '/answers': 'real',             // 回答列表
};
```

## 如何切换到全部真实服务器

当所有接口都完成后：

### 方式 1：修改全局配置（推荐）

```javascript
const USE_MOCK = false;  // 改为 false
```

然后删除或注释 `API_MODE_CONFIG`：

```javascript
const API_MODE_CONFIG = {
  // 所有接口都使用真实服务器，不需要单独配置
};
```

### 方式 2：保留配置（用于调试）

保留 `API_MODE_CONFIG`，方便以后需要时切换某个接口到 Mock：

```javascript
const USE_MOCK = false;  // 全局使用真实服务器

const API_MODE_CONFIG = {
  // 如果某个接口有问题，可以临时切换到 Mock
  // '/questions': 'mock',
};
```
