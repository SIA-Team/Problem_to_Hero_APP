## 关键统计字段

从返回的数据来看,统计相关的字段名称为:

- `likeCount`: 点赞数 (值为 0)
- `commentCount`: 评论数 (值为 0)
- `collectCount`: 收藏数 (值为 0)
- `answerCount`: 回答数 (值为 3)
- `viewCount`: 浏览数 (值为 178)

## 问题分析

**重要发现:**

1. 接口返回的字段使用的是 **驼峰命名** (camelCase):
   - `likeCount` ✅
   - `commentCount` ✅
   - `collectCount` ✅ (注意是 collect 不是 bookmark)

2. **没有以下字段:**
   - `shareCount` / `share_count` / `shares` - 转发数 ❌
   - `dislikeCount` / `dislike_count` / `dislikes` - 点踩数 ❌
   - `bookmarkCount` / `bookmark_count` / `bookmarks` - 收藏数字段名不对 ❌

3. **字段名称不匹配:**
   - 代码中使用 `bookmarkCount`,但接口返回的是 `collectCount`

## 需要修复的问题

1. 收藏数应该读取 `collectCount` 而不是 `bookmarkCount`
2. 转发数 `shareCount` 接口没有返回,应该显示 0
3. 点踩数 `dislikeCount` 接口没有返回,应该显示 0
