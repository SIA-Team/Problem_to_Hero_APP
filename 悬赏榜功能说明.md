# 悬赏榜功能说明

## 功能概述

在首页顶部导航中新增"悬赏榜"标签，展示平台上最受关注的悬赏问题排行榜。

## 设计特点

### 1. 排名展示
- 前3名使用醒目的颜色徽章（金、橙、黄）
- 第4名及以后使用灰色徽章
- 清晰的排名数字显示

### 2. 悬赏信息
- 醒目的红色悬赏金额徽章（带金币图标）
- 问题分类标签（如：编程学习、职业发展等）
- 剩余时间提示（黄色徽章）

### 3. 热度指标
- 热度值显示（带火焰图标）
- 趋势指示（上升/下降箭头）
- 浏览量统计
- 回答数统计

### 4. 问题标签
- "热"标签：红色背景，表示热门问题
- "新"标签：绿色背景，表示新发布的问题

### 5. 作者信息
- 头像显示
- 用户名展示

### 6. 交互功能
- 点击问题卡片跳转到问题详情页
- 下拉刷新更新榜单数据
- 支持翻译功能（TranslateButton）

## 排序规则

悬赏榜根据以下因素综合排名：
1. 悬赏金额（权重最高）
2. 热度值（浏览量、互动量）
3. 回答数量
4. 发布时间（新问题有加权）

## 技术实现

### 文件结构
```
src/screens/RewardRankingScreen.js  - 悬赏榜页面组件
App.js                               - 路由注册
src/screens/HomeScreen.js            - 首页导航集成
src/i18n/locales/zh.json            - 中文翻译
src/i18n/locales/en.json            - 英文翻译
```

### 主要组件
- `RewardRankingScreen`: 悬赏榜主页面
- `RewardItem`: 单个悬赏问题卡片组件
- `TranslateButton`: 翻译按钮组件（复用现有）

### 样式设计
- 采用卡片式布局
- 使用项目统一的颜色规范
- 响应式设计，适配不同屏幕尺寸

## 数据结构

```javascript
{
  id: 1,                    // 问题ID
  rank: 1,                  // 排名
  title: "问题标题",         // 问题标题
  reward: 500,              // 悬赏金额（美元）
  hot: "2856万",            // 热度值
  tag: "热",                // 标签（热/新）
  tagColor: "#ef4444",      // 标签颜色
  author: "张三丰",          // 作者名
  avatar: "头像URL",         // 作者头像
  answers: 156,             // 回答数
  viewCount: "28.5万",      // 浏览量
  timeLeft: "2天",          // 剩余时间
  isUp: true,               // 热度趋势（上升/下降）
  category: "编程学习"       // 问题分类
}
```

## 多语言支持

- 英文：Reward Ranking

1. 本功能仅添加了悬赏榜模块，未修改其他模块的样式、接口和功能

3. 可以通过热更新发布，无需重新构建应用
4. 收入榜已按要求隐藏（代码保留，注释掉）
