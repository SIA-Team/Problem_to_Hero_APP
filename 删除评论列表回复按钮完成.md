# 删除评论列表回复按钮完成

## 修改概述
在问题详情页面中，删除了补充评论列表和回答评论列表中每条评论的"回复"按钮。

## 修改位置
文件：`src/screens/QuestionDetailScreen.js`

### 1. 补充评论列表弹窗
弹窗：`showSuppCommentListModal`
位置：每条评论的操作按钮区域

### 2. 回答评论列表弹窗
弹窗：`showAnswerCommentListModal`
位置：每条评论的操作按钮区域

## 修改内容

### 删除前的代码结构：
```jsx
<View style={styles.commentListActions}>
  {/* 点赞按钮 */}
  <TouchableOpacity style={styles.commentListActionBtn}>
    <Ionicons name="thumbs-up-outline" size={14} color="#9ca3af" />
    <Text style={styles.commentListActionText}>{comment.likes}</Text>

  {/* 回复按钮 - 已删除 */}
  <TouchableOpacity style={styles.commentListActionBtn}>

  </TouchableOpacity>
</View>
```

### 删除后的代码结构：

```

```

### 修改后：
```
评论内容...
[👍 5]  [💬 2]  ← 只保留点赞和查看回复
```

## 保留的功能

### 补充评论列表：
✅ 点赞按钮（显示点赞数）
✅ 查看回复按钮（显示回复数，点击展开/收起回复列表）
❌ 回复按钮（已删除）

### 回答评论列表：
✅ 点赞按钮（显示点赞数）
✅ 查看回复按钮（显示回复数，点击展开/收起回复列表）
❌ 回复按钮（已删除）

## 用户体验改进

### 改进点：
1. **界面更简洁**：减少了一个操作按钮
2. **视觉更清爽**：操作区域更紧凑
3. **避免混淆**：用户可以通过查看回复按钮展开回复列表

### 为什么删除：
1. 用户可以通过点击评论数按钮展开回复列表
2. 减少重复的操作入口
3. 简化界面，提升用户体验

## 注意事项

### 回复列表中的回复按钮保留
在展开的回复列表中，每条回复仍然保留"回复"按钮：
```jsx
<View style={styles.replyActions}>
  <TouchableOpacity style={styles.replyActionBtn}>
    <Ionicons name="thumbs-up-outline" size={12} color="#9ca3af" />
    <Text style={styles.replyActionText}>{reply.likes}</Text>
  </TouchableOpacity>
  <TouchableOpacity style={styles.replyActionBtn}>
    <Text style={styles.replyReplyBtn}>回复</Text>  ← 这个保留
  </TouchableOpacity>
</View>
```

这是合理的，因为：
- 回复列表中的"回复"按钮用于回复具体的某条回复
- 评论列表中的"回复"按钮功能重复（可以通过展开回复列表实现）
