# 快速开始热更新

## 三步启用热更新

### 第 1 步：构建 APK（首次）
```bash
eas build --platform android --profile production
```
⏱️ 等待 10-20 分钟，构建完成后下载 APK

### 第 2 步：分发 APK
将构建好的 APK 分发给用户安装

### 第 3 步：发布更新（以后每次修改代码后）
```bash
eas update --branch production --message "修复分类显示问题"
```
✅ 用户下次打开 APP 时会自动接收更新

## 常用命令速查

```bash
# 构建 APK
eas build --platform android --profile production

# 发布更新
eas update --branch production --message "更新说明"

# 查看构建历史
eas build:list

# 查看更新历史
eas update:list

# 查看构建状态
eas build:view <build-id>
```

## 注意事项

### ✅ 可以热更新
- JavaScript 代码
- React 组件
- 样式
- 业务逻辑
- API 调用

### ❌ 需要重新构建
- 原生代码
- 原生依赖
- app.json 原生配置
- 权限变更
- SDK 升级

## 更新流程

```
修改代码 → 测试 → eas update → 用户打开 APP → 自动下载 → 重启 APP → 更新生效
```
