## 📡 接口信息

### 接口地址
```
GET /app/content/question/list
```

### 完整 URL
```
http://123.144.149.59:30560/qa-hero-content/app/content/question/list
```

### 认证方式
```
Bearer Token（必须）
```

### 请求参数
```javascript
{
  pageNum: 1,        // 页码
  pageSize: 20,      // 每页数量
  question: {        // 筛选条件
    userId: 10001,     // 提问者ID
    type: 0,           // 问题类型：0=公开，1=悬赏，2=定向
    categoryId: 5,     // 类别ID
    status: 1,         // 状态：0=草稿，1=已发布，2=已关闭
    title: "如何",     // 标题（模糊搜索）
  }
}
```

### 响应数据
```javascript
{
  "code": 200,
  "msg": "success",
  "data": {
    "total": 100,
    "rows": [
      {
        "id": 100,
        "userId": 10001,
        "type": 0,
        "title": "如何提高团队协作效率？",
        "status": 1,
        "bountyAmount": 5000,  // 单位：分
        "location": "北京市朝阳区",
        "viewCount": 1024,
        "answerCount": 3,
        "likeCount": 56,
        "createTime": "2026-03-10T00:26:41.552Z"
      }
    ]
  }
}
```

## 🔧 使用方法

### 在 HomeScreen.js 中使用

```javascript
import questionApi from '../services/api/questionApi';

// 加载推荐列表
const response = await questionApi.getRecommendList({
  pageNum: 1,
  pageSize: 20,
});

// 加载热榜列表
const response = await questionApi.getHotList({
  pageNum: 1,
  pageSize: 20,
});

// 加载关注列表
const response = await questionApi.getFollowList({
  pageNum: 1,
  pageSize: 20,
});

// 自定义筛选
const response = await questionApi.getQuestions({
  pageNum: 1,
  pageSize: 20,
  question: {
    status: 1,
    type: 0,
    title: '如何',
  },
});
```

### Token 自动处理

`contentApiClient` 会自动从 AsyncStorage 读取 Token 并添加到请求头：

```javascript
// 自动添加
Authorization: Bearer YOUR_TOKEN
```

**无需手动处理 Token！**

## ⚠️ 注意事项

### 1. Token 认证

- ✅ contentApiClient 会自动添加 Token
- ✅ 无需手动处理
- ⚠️ 确保用户已登录
- ⚠️ Token 过期需要重新登录

### 2. 金额单位

- API 返回的金额单位是**分**
- 显示时需要除以 100 转换为元

### 3. 时间格式

- API 返回 ISO 8601 格式
- 需要转换为相对时间（如"2小时前"）

### 4. 用户信息

- API 只返回 userId
- 需要获取用户名和头像
- 临时方案：使用占位符
- 长期方案：后端返回用户信息或前端批量获取

### 5. 问题类型

- 0 = 公开问题 (free)
- 1 = 悬赏问题 (reward)
- 2 = 定向问题 (targeted)
