## 📱 Android 配置总结

### 支持范围
- **最低版本**：Android 6.0 (API 23)
- **目标版本**：Android 15 (API 35)
- **覆盖率**：约 96% 的设备

### 已修复的问题

#### 1. HTTP 明文流量限制 ⚠️ 关键修复

**问题**：Android 9+ 默认禁止 HTTP 连接

**修复**：在 `AndroidManifest.xml` 中添加：
```xml
<application
  ...
  android:usesCleartextTraffic="true">
```

**文件位置**：`android/app/src/main/AndroidManifest.xml`

**状态**：✅ 已修复

---

#### 2. 媒体权限（Android 13+）

**问题**：Android 13 细化了存储权限

**修复**：在 `app.json` 中添加：
```json
"permissions": [
  "READ_MEDIA_IMAGES"
]
```

**状态**：✅ 已配置

---

### 已配置的权限

```xml
<uses-permission android:name="android.permission.INTERNET"/>
<uses-permission android:name="android.permission.CAMERA"/>
<uses-permission android:name="android.permission.READ_EXTERNAL_STORAGE"/>
<uses-permission android:name="android.permission.WRITE_EXTERNAL_STORAGE"/>
<uses-permission android:name="android.permission.READ_MEDIA_IMAGES"/>
```

## 🍎 iOS 配置总结

### 支持范围
- **最低版本**：iOS 13.4
- **目标版本**：iOS 18
- **覆盖率**：约 97% 的设备

### 已修复的问题

#### 1. App Transport Security (ATS) ⚠️ 关键修复

**问题**：iOS 9+ 默认只允许 HTTPS 连接

**修复**：在 `app.json` 中添加：
```json
"ios": {
  "infoPlist": {
    "NSAppTransportSecurity": {
      "NSAllowsArbitraryLoads": true
    }
  }
}
```

**文件位置**：`app.json`

**状态**：✅ 已修复

---

## ⚠️ 重要提醒

### 1. HTTP vs HTTPS

**当前状态**：使用 HTTP (`http://123.144.100.10:30560`)

**已配置**：
- ✅ Android 允许 HTTP
- ✅ iOS 允许 HTTP

**问题**：
- ⚠️ 数据不加密，不安全
- ⚠️ App Store 审核可能要求说明
- ⚠️ Google Play 可能有警告

**建议**：
- 🔒 **尽快升级到 HTTPS**
- 📝 审核时说明使用 HTTP 的原因
- 🎯 这是临时方案，不是长期解决方案

---

### 2. 更安全的配置方式

#### Android（推荐）

创建 `android/app/src/main/res/xml/network_security_config.xml`：

```xml
<?xml version="1.0" encoding="utf-8"?>
<network-security-config>
    <domain-config cleartextTrafficPermitted="true">
        <domain includeSubdomains="true">123.144.100.10</domain>
    </domain-config>
</network-security-config>
```

然后在 `AndroidManifest.xml` 中引用：

```xml
<application
  android:networkSecurityConfig="@xml/network_security_config">
```

**优点**：只允许特定域名使用 HTTP

---

#### iOS（推荐）

```json
"NSAppTransportSecurity": {
  "NSExceptionDomains": {
    "123.144.100.10": {
      "NSExceptionAllowsInsecureHTTPLoads": true,
      "NSIncludesSubdomains": false
    }
  }
}
```

**优点**：只允许特定域名使用 HTTP
