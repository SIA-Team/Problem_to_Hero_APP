#### 新增点踩处理函数
```javascript
const handleSupplementDislike = async (supplementId) => {
  try {
    const response = await questionApi.dislikeSupplement(supplementId);
    if (response && response.code === 200) {
      const currentState = suppDisliked[supplementId] || false;
      const newState = !currentState;
      setSuppDisliked(prev => ({ ...prev, [supplementId]: newState }));
      showToast(newState ? '已点踩' : '已取消点踩', 'success');
    }
  } catch (error) {
    showToast('网络错误，请稍后重试', 'error');
  }
};
```

## API接口

### 补充相关接口
- `POST /app/content/supplement/:id/like` - 点赞/取消点赞
- `POST /app/content/supplement/:id/dislike` - 点踩/取消点踩

### 回答相关接口
- `POST /app/content/answer/:id/like` - 点赞
- `POST /app/content/answer/:id/unlike` - 取消点赞

- `POST /app/content/answer/:id/undislike` - 取消点踩
- `POST /app/content/answer/:id/collect` - 收藏
