## 🚀 2. 缓存策略（核心）

### 2.1 多级缓存

```
请求数据流程：
1. 内存缓存（最快，1ms）
   ↓ 未命中
2. 本地存储（快，10-50ms）
   ↓ 未命中
3. 网络请求（慢，200-1000ms）
   ↓
4. 更新缓存
```

**今日头条的缓存策略**：

| 缓存类型 | 存储位置 | 有效期 | 用途 |
|---------|---------|--------|------|
| **热数据** | 内存 | 5 分钟 | 当前 Tab 数据 |
| **温数据** | AsyncStorage | 1 小时 | 相邻 Tab 数据 |
| **冷数据** | AsyncStorage | 24 小时 | 历史浏览数据 |

### 2.2 缓存更新策略

```javascript
// 今日头条的做法
1. 首次打开：显示缓存 + 后台更新
2. 下拉刷新：清除缓存 + 重新加载
3. 定时更新：每 5 分钟自动刷新
4. 智能更新：根据用户活跃度调整
```

#### 阶段 1：基础优化（1-2 天）

**1.1 接入真实 API**
```javascript
// 在 HomeScreen.js 中
import { questionApi } from '../services/api/questionApi';

const loadQuestions = async (tabType, page = 1) => {
  try {
    let data;
    switch(tabType) {
      case 'recommend':
        data = await questionApi.getRecommendList(page, 20);
        break;
      case 'hot':
        data = await questionApi.getHotList(page, 20);
        break;
      case 'follow':
        data = await questionApi.getFollowList(page, 20);
        break;
      default:
        data = await questionApi.getQuestions(page, 20);
    }
    return data;
  } catch (error) {
    console.error('加载失败:', error);
    return [];
  }
};
```

**1.2 添加基础缓存**
```javascript
import AsyncStorage from '@react-native-async-storage/async-storage';

// 缓存数据
const cacheData = async (key, data) => {
  try {
    await AsyncStorage.setItem(
      `cache_${key}`,
      JSON.stringify({
        data,
        timestamp: Date.now()
      })
    );
  } catch (error) {
    console.error('缓存失败:', error);
  }
};

// 读取缓存
const getCachedData = async (key, maxAge = 5 * 60 * 1000) => {
  try {
    const cached = await AsyncStorage.getItem(`cache_${key}`);
    if (!cached) return null;
    
    const { data, timestamp } = JSON.parse(cached);
    const age = Date.now() - timestamp;
    
    if (age > maxAge) {
      // 缓存过期
      return null;
    }
    
    return data;
  } catch (error) {
    console.error('读取缓存失败:', error);
    return null;
  }
};
```

**1.3 优化加载流程**
```javascript
const loadQuestionsWithCache = async (tabType, page = 1) => {
  const cacheKey = `${tabType}_${page}`;
  
  // 1. 先显示缓存
  const cached = await getCachedData(cacheKey);
  if (cached && page === 1) {
    setQuestionList(cached);
  }
  
  // 2. 后台加载最新数据
  const fresh = await loadQuestions(tabType, page);
  
  // 3. 更新显示和缓存
  if (fresh && fresh.length > 0) {
    if (page === 1) {
      setQuestionList(fresh);
    } else {
      setQuestionList(prev => [...prev, ...fresh]);
    }
    await cacheData(cacheKey, fresh);
  }
};
```
