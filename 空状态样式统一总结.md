# 空状态样式统一总结

## 问题描述
回答详情页面和问题详情页面的暂无数据样式不一致，需要统一样式规范。

## 统一标准
以问题详情页面的空状态样式为标准：

### 样式规范
- **容器**: 居中对齐，垂直内边距 60px
- **图标**: 48px 大小，颜色 #d1d5db
- **标题**: 16px 字体，600 字重，颜色 #6b7280，顶部边距 16px
- **描述**: 14px 字体，颜色 #9ca3af，顶部边距 8px

## 实现方案

### 1. 创建通用组件
**文件**: `src/components/EmptyState.js`
- 创建可复用的空状态组件
- 支持自定义图标、标题、描述
- 使用统一的样式规范

### 2. 更新回答详情页面
**文件**: `src/screens/AnswerDetailScreen.js`

#### 补充回答空状态
```jsx
<EmptyState
  icon="chatbubbles-outline"
  title="暂无补充回答"
  description="还没有人发布补充回答"
/>
```

#### 评论空状态
```jsx
<EmptyState
  icon="chatbubble-outline"
  title="暂无评论"
  description="快来发表第一条评论吧"
/>
```

### 3. 样式清理
- 移除旧的空状态样式定义
- 统一使用 EmptyState 组件
- 保持代码简洁性

- 回答详情页面 - 补充回答列表

## 使用指南

### 基础用法
```jsx
import EmptyState from '../components/EmptyState';

<EmptyState
  icon="chatbubble-outline"
  title="暂无数据"
  description="还没有相关内容"
/>
```

### 自定义用法
```jsx
<EmptyState
  icon="document-outline"
  title="暂无文档"
  description="快来上传第一个文档吧"
  iconSize={56}
  iconColor="#3b82f6"
/>
```
