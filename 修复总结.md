# 应用错误修复总结

## 问题描述
应用在模拟器中报错：`TypeError: Cannot read property 'getRecommendList' / 'getQuestions' of undefined`

## 根本原因
`src/utils/dataLoader.js` 中的 API 导入方式错误：
- **错误**：`import { questionApi } from '../services/api/questionApi';` （命名导入）
- **正确**：`import questionApi from '../services/api/questionApi';` （默认导入）

因为 `questionApi.js` 使用的是 `export default`，而不是命名导出。

## 已完成的修复

### 1. ✅ 修复 API 导入
**文件**：`src/utils/dataLoader.js`
**修改**：
```javascript
// 修改前
import { questionApi } from '../services/api/questionApi';

// 修改后
import questionApi from '../services/api/questionApi';
```
