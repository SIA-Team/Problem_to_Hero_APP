## 前端实现

### 状态管理
```javascript
const [suppDisliked, setSuppDisliked] = useState({});           // 点踩状态
const [suppDislikeLoading, setSuppDislikeLoading] = useState({}); // 加载状态
const [suppDislikeDebounce, setSuppDislikeDebounce] = useState({}); // 防抖定时器
```

### 核心逻辑
```javascript
const handleSupplementDislike = async (supplementId) => {
  // 1. 参数验证
  if (!supplementId) return;
  
  // 2. 防重复请求
  if (suppDislikeLoading[supplementId]) return;
  
  // 3. 清除防抖定时器
  if (suppDislikeDebounce[supplementId]) {
    clearTimeout(suppDislikeDebounce[supplementId]);
  }
  
  // 4. 立即更新UI（乐观更新）
  const currentState = suppDisliked[supplementId] || false;
  const newState = !currentState;
  setSuppDisliked(prev => ({ ...prev, [supplementId]: newState }));
  
  // 5. 防抖延迟执行
  const debounceTimer = setTimeout(async () => {
    try {
      // 设置加载状态
      setSuppDislikeLoading(prev => ({ ...prev, [supplementId]: true }));
      
      // 根据状态调用不同接口
      const response = newState 
        ? await questionApi.dislikeSupplement(supplementId)
        : await questionApi.undislikeSupplement(supplementId);
      
      // 处理响应
      if (response?.code === 200) {
        showToast(newState ? '已点踩' : '已取消点踩', 'success');
      } else {
        // 失败回滚状态
        setSuppDisliked(prev => ({ ...prev, [supplementId]: currentState }));
        showToast(response?.msg || '操作失败', 'error');
      }
    } catch (error) {
      // 错误回滚状态
      setSuppDisliked(prev => ({ ...prev, [supplementId]: currentState }));
      showToast('网络错误，请稍后重试', 'error');
    } finally {
      // 清理状态
      setSuppDislikeLoading(prev => ({ ...prev, [supplementId]: false }));
      setSuppDislikeDebounce(prev => {
        const newState = { ...prev };
        delete newState[supplementId];
        return newState;
      });
    }
  }, 300);
  
  // 保存防抖定时器
  setSuppDislikeDebounce(prev => ({ ...prev, [supplementId]: debounceTimer }));
};
```

### UI实现
```javascript
<TouchableOpacity 
  style={[
    styles.suppActionBtn,
    suppDislikeLoading[item.id] && styles.suppActionBtnLoading
  ]}
  onPress={(e) => { 
    e.stopPropagation(); 
    handleSupplementDislike(item.id);
  }}
  disabled={suppDislikeLoading[item.id]}
>
  <Ionicons 
    name={suppDisliked[item.id] ? "thumbs-down" : "thumbs-down-outline"} 
    size={16} 
    color={suppDisliked[item.id] ? "#ef4444" : "#6b7280"} 
  />
  <Text style={[styles.suppActionText, suppDisliked[item.id] && { color: '#ef4444' }]}>
    {item.dislikes + (suppDisliked[item.id] ? 1 : 0)}
  </Text>
  {suppDislikeLoading[item.id] && (
    <ActivityIndicator size="small" color="#ef4444" style={styles.suppActionLoader} />
  )}
</TouchableOpacity>
```

### 状态初始化
```javascript
// 在补充列表加载时初始化点踩状态
const initialDislikedState = {};
newSupplements.forEach(item => {
  initialDislikedState[item.id] = item.isDisliked || item.disliked || false;
});
setSuppDisliked(prev => ({ ...prev, ...initialDislikedState }));
```

## 功能特性

### 1. 状态切换逻辑
- **未点踩 → 点踩**: 调用 `dislikeSupplement(id)`
- **已点踩 → 取消点踩**: 调用 `undislikeSupplement(id)`

### 2. 防抖机制
- 300ms防抖延迟，避免频繁点击
- 自动清理防抖定时器

### 3. 防重复请求
- 请求进行中时禁用按钮
- 忽略重复点击

### 4. 乐观更新
- 立即更新UI状态，提供即时反馈
- 请求失败时自动回滚状态

### 5. 错误处理
- 网络错误处理
- 服务器错误处理
- 用户友好的错误提示

### 6. 加载状态
- 显示加载指示器
- 按钮禁用状态

## 接口规范

### 点踩接口
- **URL**: `POST /app/content/supplement/{id}/dislike`
- **参数**: `id` (路径参数, int类型, 必传, 补充ID)
- **返回**: `{"code": 200, "msg": "", "data": {}}`

### 取消点踩接口
- **URL**: `POST /app/content/supplement/{id}/undislike`
- **参数**: `id` (路径参数, int类型, 必传, 补充ID)
- **返回**: `{"code": 200, "msg": "", "data": {}}`
