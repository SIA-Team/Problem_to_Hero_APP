## 核心优化点

### 1. 缓存优先加载
```javascript
// 立即尝试显示缓存
const cachedData = await loadData(tabType, 1, false);
if (cachedData.length > 0) {
  console.log(`⚡ 立即显示缓存数据: ${cachedData.length} 条`);
  setQuestionList(cachedData); // 立即显示
  
  // 后台静默更新
  setTimeout(() => {
    silentUpdate(tabType);
  }, 100);
}
```

### 2. 防抖机制
```javascript
// 避免频繁刷新
const now = Date.now();
if (now - lastRefreshTime.current < 1000) {
  console.log('⏱️ 刷新太频繁，已忽略');
  return;
}
```

### 3. 最小显示时间
```javascript
// 避免闪烁
setTimeout(() => {
  setRefreshing(false);
}, 300);
```

### 4. 延迟预加载
```javascript
// 不阻塞主请求
setTimeout(() => {
  if (allTabs.length > 0) {
    prefetchAdjacentTabs(activeTab, allTabs);
  }
}, 500);
```

## 常见问题

### Q: 如果没有缓存怎么办？
A: 会自动降级到网络加载，显示 loading 状态。

### Q: 缓存数据会过期吗？
A: 会的。后台会自动更新缓存，保持数据新鲜。

### Q: 如何清除缓存？
A: 可以在设置中添加"清除缓存"功能，或者应用会自动管理缓存。

### Q: 会占用多少存储空间？
A: 约 1-5MB，会自动清理旧数据。
